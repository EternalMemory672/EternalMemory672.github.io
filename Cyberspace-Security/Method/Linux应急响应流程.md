## Linux应急响应流程
#Linux #应急响应
### 1. 入侵排查思路
#### 1.1 账号安全
基本使用：
1. 用户信息文件：`/etc/passwd`
	```
	account:password:UID:GID:GECOS:directory:shell
	用户名：密码：用户ID：组ID：用户说明：家目录：登录shell
	```
	**无密码只允许本地登陆**
2. 影子文件：`/etc/shadow`
	```
	用户名：加密密码：最后一次修改时间：最小修改时间间隔：密码有效期：密码需要变更前的警告天数：密码过期后的宽限时间：账号失效时间：保留字段
	```
3. 常用命令
	`who` 查看当前登录用户（tty本地登陆、pts远程登录）
	`w` 查看系统信息、某时刻用户行为
	`uptime` 查看登陆多久、多少用户、负载
