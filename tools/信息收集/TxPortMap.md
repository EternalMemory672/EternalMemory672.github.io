## TxPortMap
#信息收集 #端口扫描
win/kali
```
  -ei value
        set ip ranges to exclude
  -ep value
        set port ranges to exclude
  -i value
        set domain and ips
  -json
        output json format
  -l string
        input ips file
  -limit int
        number of goroutines, between 1 and 2000
  -n int
        number of goroutines, between 1 and 2000 (default 800)
  -nbtscan
        get netbios stat by UDP137 in local network
  -nocolor
        using color ascii to screen
  -o string
        success log (default "rst.txt")
  -p value
        set port ranges to scan，default is top100
  -r    random scan flag
  -t float
        tcp connect time out default 0.5 second (default 0.5)
  -t1000
        scan top1000 ports
  -tracefile string
        request log
```
